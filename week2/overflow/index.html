over property discription

The overflow property specify whether to click or to add scroll bar when the content of an element is too big to fit a specify area

The overflow property has the following properties which are

  • Visible
  • Hidden
  • scroll
  • Auto

NOTE,the overflow properties only work for block element with specified height

NOTE, in OXXLION scroll barsare Hidden by defult even though overflow is set to scrool

overflow-X and oveflow-Y

The overflow-X and overflow-Y properties specify whether to change the overflow content just verticaly or horizontaly or both

overfflow-x specifies what to do withthe left/right edges of the content

oveflow-Y specifies what to do with the top/bottum edges of the content



You can use the overflow property when you want to have better control over the layout.The overflow propety specifies what happen when the content overflow an element
