Responsive web design(RWD)

Responsive web design(RWD) is a design approach that ensures a website looks and functions well across a wide range of design - From mobile phones to tablets to dektop computer - without requiering seprate version of the site

Key concept of responsive web design

  1. fluidgrids:

    Layouts are built using reletive units like percentages allowing elemnts to reside base on the screen.

  2. Flexible images:

    images scale within their containing elemnts so they do not overflow or distort on different screen sizes

  3. Media quarre:

    css rules tht aply styles base on the device characertics.

    Example
    2:44 
    
                      @media (max-width: 768px) {

  4. mobile-first approach:

    design start with the smallest screen size and scales up ensuring a good user experience on mobile devices first.
