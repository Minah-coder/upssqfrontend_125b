CSS font-family property

The css font-family property specifies the font of the text

Note:if the font name is more than one word it must be in qoatation mark, like:"times new roman"

Tip: the font-family property should hold several font name as a fall back system .the browser will try to use the first one if it is not avalable it will use the next one

example

.p1{
font-family property:times new roman;
}
.p2{
font-family property:arial helvetical sans-serif;
}
.p3{
font-family property:lusidial-consol erial monospace;
}
